# Session Memories

This directory stores session summaries for continuity across conversations.

## Files

- `latest.md` - Most recent session summary (auto-updated by /reflect)
- `YYYY-MM-DD.md` - Date-stamped session archives

## Usage

Use `/reflect resume` to load the latest session and continue where you left off.

## Template

See `.claude/templates/session-summary.md` for the session summary template.
